Load Firmware Update From File

Check Midi Status

Explanation:

Press the Alive button to test if the USBMIDI is connected and which version is running.

To the left you find the Console which logs actions. If you see no response your USBMIDI is not connected.

When Updating please be very sure that the file you upload is correct ! If you upload a invalid file the module will update to it regardless, but will not be able to run correctly and will NOT be able to revert or update to a correct version

As soon as you select a file it will try to upload it. If the name of the file doesn't match: "USBMIDI.bin" then it will not try to update the module.
