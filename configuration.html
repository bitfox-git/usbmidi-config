Explanation:

Press the Alive button to test if the USBMIDI is connected and which version is running.

To the left you find the Console which logs actions. If you see no response your USBMIDI is not connected.

When done configuring, send the configuration to USBMIDI and save the configuration to Flash. If you don't save to Flash the new configuration settings will be wiped on the next power cycle.

Please note: Keypressure is currently implemented the same as Velocity

If you want to use the Polyphonic modes, you have to start at the first output (or third in the duo mode). So if you want to use 3 voices in quad mode use output 1 2 and 3.
