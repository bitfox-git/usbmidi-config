Upload / Download / Save User Calibration Data from USBMIDI

Explanation:

Press the Alive button to test if the USBMIDI is connected and which version is running.

To the left you find the Console which logs actions. If you see no response your USBMIDI is not connected.

Use a Multimeter and write the value you read in the corresponding input field.

When done calibrating, send the calibration to USBMIDI and save the calibration to Flash. If you don't save to Flash the new calibration settings will be wiped on the next power cycle.
